A Taste of D3

Trying to teach all of D3 in one class is pretty hard.
To really understand D3, you need to know:

  • HTML
  • CSS
  • SVG
  • JavaScript
  • The D3 JavaScript library

It's particularly tricky if you're trying to figure out how to use D3 to do things that Excel, Tableau, etc can't do.

So rather than trying to cover all of D3, the class "A Taste of D3" tries to give students a quick intro to D3 and a feel for what it can do -- and what's easy and hard to do.

This website has the sample code and notes for the class.
